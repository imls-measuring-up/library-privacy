Library and collections

Search WorldCat for books, articles, etc..

You can use WorldCat to find books, journals, articles, and other resources, in both print or electronic format. Your search will cover McGill's entire collection of resources, and can be expanded to include the collections of libraries worldwide.

If we don't have the item at McGill, you'll be able to request the item using interlibrary loan.

Advanced search Classic Catalogue

You can use the Classic Catalogue to search for print books, journals, DVDs, and other items that are in McGill's collection.
